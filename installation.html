Installation¶

Stable release¶

To install SteganoGAN, run this command in your terminal:

$ pip install steganogan

This is the preferred method to install SteganoGAN, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From source¶

The source for SteganoGAN can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/DAI-Lab/SteganoGAN

Or download the tarball:

$ curl  -OL https://github.com/DAI-Lab/SteganoGAN/tarball/master

Once you have a copy of the source, you can install it with:

$ make install
